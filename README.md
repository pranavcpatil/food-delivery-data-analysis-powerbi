# 📊 Food Delivery Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project focuses on analyzing food delivery data using Power BI to extract meaningful insights and support business decisions.

---

## 🎯 Objectives

* Analyze customer ordering behavior
* Identify top-selling food items
* Evaluate restaurant performance
* Understand category-wise trends

---

## 📈 Key Insights

* Samosa is the most ordered food item
* Main Course category dominates overall orders
* Bukhara generates the highest revenue
* A small number of customers contribute to most orders

---

## 📊 Dashboard Features

* KPI Cards (Total Orders, Quantity, Customers, Restaurants)
* Top Selling Food Items (Bar Chart)
* Category-wise Orders (Donut Chart)
* Top Restaurants by Revenue (Bar Chart)
* Orders by Restaurant (Column Chart)
* Top Customers by Orders (Bar Chart)
* Interactive Slicers (Category, Restaurant, Order Status, Date)
